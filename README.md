# 🌙 RizqAnalyser

Universal job screening extension for Muslims.

## 🌟 Features
- **Multi-Platform Detection**: Reliable company and role extraction from LinkedIn, Indeed, and Glassdoor.
- **Hybrid Analysis**: Combines Wikidata SPARQL queries with a comprehensive rule-based keyword engine.
- **Scriptural Grounding**: Provides authenticated verses and hadiths for every verdict.
- **Zero-AI Privacy**: No LLMs or external AI APIs; analysis is purely rule-based and transparent.

## 🛠️ Installation
1. Download this repository.
2. Go to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the `RizqAnalyser` folder.

## 🏗️ Architecture
`Detection` $\rightarrow$ `Wikidata` $\rightarrow$ `Keyword Rules` $\rightarrow$ `Scripture Mapping`

## ⚠️ Disclaimer
**This tool is NOT a Fatwa.** It provides a preliminary screening based on keywords. Final rulings belong to qualified Islamic scholars. Please consult a Mufti for a definitive ruling.
